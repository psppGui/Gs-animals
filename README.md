
#  GS Animals

Aplicativo desktop desenvolvido com **Electron** para auxiliar **veterinários e tutores de pets** na consulta de informações sobre animais.

---

##  Sobre o projeto

O **GS Animals** é um aplicativo simples e intuitivo que permite:

*  Consultar possíveis problemas de saúde
*  Identificar características de raças
*  Obter informações sobre comportamento animal

---

##  Tecnologias utilizadas

* Electron
* JavaScript
* HTML5
* CSS3

---

##  Como executar o projeto

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/gs-animals.git

# Acesse a pasta
cd VetApp

# Instale as dependências
npm install

# Execute o app
npm start
```
